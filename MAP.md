# Winchester Meshtastic Network Map

Our network is documented on an interactive map with three layers showing current nodes, distance between nodes, and possible future nodes.

**[View the Winchester Meshtastic Map](https://www.google.com/maps/d/viewer?mid=1wNGFs7jZIJAwrGD2tcmaawHkiA1TW-E)**

## Map Layers

| Layer | What It Shows |
|-------|---------------|
| **Current Nodes** | Active node locations with short name and device hardware |
| **Node Distances** | Measured straight path between nodes |
| **Potential Future Nodes** | Future expansion candidates under consideration |
