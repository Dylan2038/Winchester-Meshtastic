# Winchester Meshtastic
A new network for all of Winchester to use for text based communication. Built by the community for the community to be open and free to use as a local communications network.

## About the Network
Currently no overall coverage of the city but some nodes around the west and north sides of Winchester can sometimes see each other. Working towards finding solid anchor locations to establish reliable relays.

## Getting Involved
Meshtastic is free, open-source mesh networking that runs on inexpensive LoRa hardware

Anyone interested can join and get involved by buying supported hardware and coordinating with the Open Source Winchester community. Ask us on Facebook or Discord for help!
If you are elevated on the edge of Winchester you may be a prime candidate for an anchor node.
